# Product Overview

**testemu** is a Flutter-based job finder mobile application that connects job seekers with employers. The app provides a comprehensive platform for job searching, application management, and professional networking.

## Key Features

- **Authentication System**: Complete user registration, login, password management, and OTP verification
- **Job Management**: Job posting, searching, application tracking (approved, pending, cancelled jobs)
- **Messaging System**: Real-time chat functionality between job seekers and employers using Socket.IO
- **Profile Management**: User profiles with personal and work information
- **Notifications**: Push notification system for job updates and messages
- **Multi-platform Support**: Built for Android, iOS, Web, Windows, macOS, and Linux
- **Location Services**: Google Maps integration for location-based job searching
- **Document Management**: Resume creation, business document uploads, and file handling
- **AI Tools**: Integrated AI features for job matching and recommendations
- **WhatsApp Integration**: Direct communication support through WhatsApp

## User Types

- **Job Seekers**: Can search jobs, apply, manage applications, create profiles
- **Employers**: Can post jobs, review applications, manage business profiles, verify accounts

## Core Functionality

The app follows a feature-based architecture with separate modules for authentication, messaging, notifications, profile management, and settings. It includes comprehensive job application tracking, salary comparison tools, and appointment scheduling features.
